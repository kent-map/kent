## Hosted (HTML5-compatible) video

Typically a **.mp4**, **.ogg** or **.webm** file.  In this example a **.webm** video file hosted by Wikimedia Commons is used.

`ve-video src=gh:kent-map/images/videos/Dickens_Museum.mp4`
